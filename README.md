# Dashboard-Control-Gestion-Retail
Desarrollé un dashboard gerencial dividido en visión ejecutiva y operativa para el monitoreo de KPIs comerciales (Ticket Promedio, Unidades Vendidas, Cumplimiento de Metas).
Implementé mapas de calor para evaluar el rendimiento mensual de las distintas sucursales e integré análisis de correlación (Ventas vs Metros Cuadrados) para medir la eficiencia por tienda.
Impacto: Permitió identificar de forma rápida oportunidades de mejora operativa, tendencias de consumo por día de la semana y el mix de medios de pago.
